# Ghost theme

Developer friendly theme for [Ghost](https://ghost.org/).

## Tech

-   Postcss + TailwindCss
-   Parcel
-   Svelte

## Usage

Clone / fork and `npm install`, then

```
npm run build
npm run build-js
npm run build-css
npm run watch-js
npm run watch-css
npm run zip
```

Pretty modular and tweakable / customizable for a dev.
